# Iraq Conflict Data Analysis

## Overview
This project analyzes terrorist incidents in Iraq to understand patterns, trends, and factors influencing the severity of attacks.

## Objectives
- Analyze trends in attack frequency over time  
- Study the distribution and outliers of casualties  
- Identify common attack and target types  
- Examine relationships between killed, wounded, and total casualties  
- Perform statistical testing (t-test)  
- Build a regression model to analyze casualty drivers  

## Tools & Technologies
- Python  
- Pandas, NumPy  
- Matplotlib, Seaborn  
- Scikit-learn  
- SciPy  

## Key Insights
- Casualties are highly skewed, with a few extreme events driving most of the impact  
- Suicide attacks result in significantly higher casualties  
- Strong positive correlation exists between fatalities and injuries  
- Certain attack and target types are more dominant  

## Files
- `main.ipynb` → Complete analysis notebook  
- `iraq_conflict_data.csv` → Cleaned dataset  

## How to Run
1. Open `main.ipynb` in Jupyter Notebook  
2. Run all cells to reproduce the analysis
